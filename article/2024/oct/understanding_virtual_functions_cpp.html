Understanding Virtual Functions in C++

Introduction

A virtual function in C++ is a function that is defined in the base class and is intended to be overridden in derived classes. Virtual functions allow for dynamic binding, meaning that the function that gets called depends on the type of object pointed to, not the type of the pointer. This feature is a key part of achieving polymorphism in object-oriented programming.

Example of Virtual Functions

Below is an example demonstrating virtual functions in C++. Each Marvel hero is a class, and they perform different actions when the action method is called. Despite the different actions, they all share the same method signature, allowing polymorphism.


#include <iostream>

class MarvelHero {
public:
    // Virtual function to allow dynamic dispatch
    virtual void action() {
        std::cout << "The hero is ready for action!" << std::endl;
    }
    virtual ~MarvelHero() {} // Virtual destructor
};

class IronMan : public MarvelHero {
public:
    // Override action() method for Iron Man
    void action() override {
        std::cout << "Iron Man shoots repulsor beams!" << std::endl;
    }
};

class Hulk : public MarvelHero {
public:
    // Override action() method for Hulk
    void action() override {
        std::cout << "Hulk smashes everything!" << std::endl;
    }
};

class CaptainAmerica : public MarvelHero {
public:
    // Override action() method for Captain America
    void action() override {
        std::cout << "Captain America throws his shield!" << std::endl;
    }
};

int main() {
    // Create objects of derived classes
    MarvelHero* hero1 = new IronMan();
    MarvelHero* hero2 = new Hulk();
    MarvelHero* hero3 = new CaptainAmerica();

    // Call action method on each hero (demonstrates polymorphism)
    hero1->action(); // Iron Man's action
    hero2->action(); // Hulk's action
    hero3->action(); // Captain America's action

    // Clean up
    delete hero1;
    delete hero2;
    delete hero3;

    return 0;
}

This code demonstrates how a virtual function works in C++. Each hero class overrides the `action()` method to perform a different action, but all use the same base class pointer to call the appropriate function. This is the essence of polymorphism in C++.
